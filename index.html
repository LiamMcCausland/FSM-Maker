How to Use the FSM Builder

  • Double-click on empty space to add a new state.
  • Shift + double-click a state to toggle it as an accepting state (double-ringed).
  • Left-click a state to select it and start typing to change its label.
  • Right-click on a state, then right-click on another state to create a transition. Self-loops are created by right-clicking the same state twice.
  • Click the "Clear" button to reset the diagram.
  • Subscripts, Greek letters, →, and ∪ are supported in labels. Example: q_0, alpha, ->, sqcup
